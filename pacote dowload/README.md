# Cardápio Digital - Lanchonete

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Selecione os itens do cardápio clicando em "Adicionar"
3. Ajuste as quantidades no carrinho usando os botões + e -
4. Clique em "Enviar Pedido via WhatsApp" para enviar

## Configuração do WhatsApp

Para configurar o número de WhatsApp da lanchonete:

1. Abra o arquivo `script.js`
2. Localize a linha: `const numeroWhatsApp = '5524992525931';`
3. Substitua pelo número desejado no formato: código do país + DDD + número (sem espaços ou caracteres especiais)
4. Exemplo: `5511987654321` para um número de São Paulo

## Instalação como App no Smartphone

### Android (Chrome):
1. Abra o site no Chrome
2. Toque no menu (3 pontos)
3. Selecione "Instalar app" ou "Adicionar à tela inicial"
4. O app será instalado como se fosse um app nativo

### iOS (Safari):
1. Abra o site no Safari
2. Toque no botão de compartilhar (quadrado com seta)
3. Role para baixo e toque em "Adicionar à Tela de Início"
4. Toque em "Adicionar"

### Recursos PWA (Progressive Web App):
- Funciona offline após primeira visita
- Ícone na tela inicial
- Experiência de app nativo
- Notificações de instalação automáticas
- Cache inteligente para carregamento rápido

## Hospedagem

Você pode hospedar os arquivos em qualquer serviço de hospedagem gratuita:
- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting

**Importante:** Para o PWA funcionar completamente, o site precisa estar em HTTPS (todas as hospedagens acima oferecem HTTPS gratuito).

Depois de hospedado, compartilhe o link diretamente com seus clientes!
